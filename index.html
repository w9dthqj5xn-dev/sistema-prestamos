📱 Instalar como Aplicación

Instala el sistema como una aplicación de escritorio o móvil para un acceso rápido.

Cómo instalar:

  • Windows/Mac: Click en el botón "Instalar Aplicación" o presiona Ctrl+D (Windows) / Cmd+D (Mac) en el navegador
  • Android: Toca el menú ⋮ y selecciona "Instalar aplicación"
  • iPhone: Toca compartir y selecciona "Agregar a pantalla de inicio"

⚠️ Zona de Peligro

Esta acción eliminará todos los datos del sistema de forma permanente.
