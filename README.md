# Sistema de Prestamos

Sistema completo para la gestión de prestamos con cálculo de intereses, seguimiento de clientes y control de pagos.

## 🚀 Características

- **Calculadora de Prestamos**: Calcula pagos mensuales, interés total y genera tabla de amortización
- **Gestión de Clientes**: Registra y administra información de clientes con sus préstamos
- **Control de Pagos**: Registra pagos realizados y visualiza pagos pendientes
- **Persistencia de Datos**: Toda la información se guarda localmente en el navegador
- **Interfaz Moderna**: Diseño responsive y fácil de usar

## 📋 Funcionalidades

-### Calculadora de Prestamos
- Calcula el pago mensual basado en monto, tasa de interés y plazo
- Genera tabla de amortización detallada mes por mes
- Muestra el total a pagar y el interés total

### Gestión de Clientes
- Registra clientes con información completa
- Visualiza el estado de cada préstamo
- Barra de progreso visual de pagos
- Búsqueda de clientes por nombre o teléfono
- Opción de ver detalle completo e imprimir

### Control de Pagos
- Registra pagos de cualquier cliente
- Historial completo de todos los pagos
- Filtrado de pagos por cliente
- Notas adicionales en cada pago

## 🛠️ Instalación y Uso

1. **Abrir el sistema**: 
   - Simplemente abre el archivo `index.html` en tu navegador web
   - No requiere instalación de servidor ni dependencias

2. **Comenzar a usar**:
   - Ve a la pestaña "Calculadora" para hacer cálculos rápidos
   - Ve a "Clientes" para registrar nuevos prestamos
   - Ve a "Pagos" para registrar los abonos de los clientes

## 💾 Almacenamiento de Datos

Los datos se guardan automáticamente en el almacenamiento local del navegador (localStorage). Esto significa que:
- La información persiste incluso si cierras el navegador
- Los datos solo están disponibles en el mismo navegador y computadora
- Para hacer respaldo, puedes exportar los datos del localStorage

## 📱 Responsive

El sistema está completamente optimizado para dispositivos móviles, tablets y escritorio.

## 🔒 Privacidad

Todos los datos se almacenan localmente en tu navegador. No se envía ninguna información a servidores externos.

## 📖 Guía de Uso

### Calcular un Préstamo
1. Ve a la pestaña "Calculadora"
2. Ingresa el monto, tasa de interés mensual y plazo
3. Haz clic en "Calcular"
4. Revisa los resultados y la tabla de amortización

### Registrar un Cliente
1. Ve a la pestaña "Clientes"
2. Llena todos los datos del formulario
3. Ingresa el monto, tasa y plazo del préstamo
4. Haz clic en "Registrar Cliente"

### Registrar un Pago
1. Ve a la pestaña "Pagos"
2. Selecciona el cliente que realizó el pago
3. Ingresa el monto y la fecha del pago
4. Opcionalmente agrega notas
5. Haz clic en "Registrar Pago"

### Ver Detalle de un Cliente
1. En la lista de clientes, haz clic en "Ver Detalle"
2. Se abrirá una nueva ventana con toda la información
3. Puedes imprimir el detalle desde esa ventana

## 🎨 Personalización

Puedes modificar los colores y estilos editando el archivo `styles.css`. Las variables CSS están definidas al inicio del archivo para fácil personalización.
